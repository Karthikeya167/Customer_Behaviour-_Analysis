# Customer Behaviour Analysis using SQL

## 📌 Project Overview
This project analyzes customer purchase behavior using SQL. It helps understand sales trends, customer preferences, repeat purchases, returns, reviews, and loyalty patterns. The goal is to generate useful business insights from raw transactional data.

## 🛠️ Tools & Technologies
- MySQL
- SQL Workbench
- SQL Queries
- Joins
- CTEs
- Window Functions
- Aggregate Functions

## 📂 Features
- Analyze total customers, orders, and revenue
- Identify top customers by spending
- Find best-selling products and categories
- Monthly sales and growth trends
- Customer retention and repeat buyers
- Return and refund analysis
- Review sentiment analysis
- Coupon usage impact
- Payment method preferences

## 📊 Key Insights
- Top-performing products increase revenue
- Loyal customers contribute more sales
- Repeat customers improve retention
- Returns affect profitability
- Coupons influence buying behavior

## ▶️ How to Run
1. Open MySQL Workbench
2. Import the `.sql` file
3. Execute queries step by step
4. View results and insights

## 🎯 Outcome
This project helps businesses make data-driven decisions to improve sales, customer satisfaction, and retention.
